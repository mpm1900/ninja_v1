# Pain Paths: Playstyles, Roles, and Signature Moves

## Deva Path (Leader/Controller)
- **Role:** Field control, burst damage, disruption
- **Playstyle:** Manipulates battlefield, disables enemies, high-impact moves
- **Signature Moves:**
  - Shinra Tensei (field-wide push/damage)
  - Chibaku Tensei (trap/immobilize)
  - Banshō Ten’in (pull enemy)

## Animal Path (Summoner/Support)
- **Role:** Summons creatures, field presence, utility
- **Playstyle:** Floods field with summons, provides cover, disrupts
- **Signature Moves:**
  - Multi-Summon (multiple creatures)
  - Chameleon Cloak (stealth)
  - Giant Dog (persistent summon)

## Asura Path (Bruiser/Disruptor)
- **Role:** Burst damage, AoE, anti-support
- **Playstyle:** High offense, disrupts formations, self-destruct options
- **Signature Moves:**
  - Missile Barrage
  - Chakra Cannon
  - Self-Destruct

## Human Path (Debuffer/Finisher)
- **Role:** Debuffs, executes weakened foes, info control
- **Playstyle:** Steals buffs, disables, finishes low-HP targets
- **Signature Moves:**
  - Soul Extraction (insta-KO on low HP)
  - Mind Probe (reveal enemy moves)
  - Chakra Drain

## Naraka Path (Healer/Reviver)
- **Role:** Healing, revival, status cleanse
- **Playstyle:** Restores allies, removes debuffs, punishes attackers
- **Signature Moves:**
  - King of Hell (revive ally)
  - Judgement (punish debuffed enemies)
  - Interrogation (stun)

## Preta Path (Tank/Absorber)
- **Role:** Damage soak, chakra absorption, anti-caster
- **Playstyle:** Absorbs jutsu, reflects status, protects team
- **Signature Moves:**
  - Chakra Absorption
  - Barrier Dome
  - Status Reflection

## Synergy Ideas
- “Six Paths Unification”: Bonus if multiple Pains are present
- Shared field effects or combo moves
